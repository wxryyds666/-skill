# -skill
一套基于思维蒸馏技术的投研数字人能力套件，核心是复刻真人股票投顾的思考框架与表达人格，而非简单堆砌知识素材。 核心特性 一键思维蒸馏：投喂研报、直播文稿、交易复盘等素材，执行 /学习 自动完成蒸馏。依托三重验证标准，分层提取心智模型、决策启发式、表达 DNA，永久写入本地记忆文件。 专业投研底座：内置基本面、行业研究、宏观周期、仓位风控完整分析框架；问答融合固化思维与实时行情资讯，信息分级鉴伪，输出明确观点与实操建议。 真人交互体验：全程口语化聊天风格，屏蔽 AI 套话与免责话术，全品类话题可承接，还原有偏好、有盲区、有独立判断的真人投顾质感。 支持增量记忆迭代，可快速复刻不同投资风格数字人，适用于智能投顾、投教直播、行情陪伴场景。
